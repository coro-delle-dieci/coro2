Come formattare il testo dei canti

Questa guida spiega come strutturare il testo dei canti per essere correttamente convertiti in HTML dallo script.

Struttura base

Separa le diverse sezioni del canto (strofe, ritornelli, ecc.) con una riga vuota:

Prima strofa
seconda riga
terza riga

Seconda strofa
seconda riga
terza riga

Sezioni speciali

Puoi specificare diversi tipi di sezioni usando i seguenti prefissi:

  • Rit.: - Per il ritornello
  • intro: - Per l'introduzione
  • outro: - Per la conclusione
  • bridge: - Per un bridge
  • Rit. - Da sola su una riga per ripetere l'ultimo ritornello
intro: Introduzione del canto

Prima strofa
prima riga
seconda riga

Rit.: Ritornello principale
prima riga del rit
seconda riga del rit

Seconda strofa
prima riga Rit.

(verrà mostrata la strofa seguita dal ritornello)

Formattazione del testo

Il testo verrà formattato come segue:

  • Le strofe saranno formattate come testo normale (come questo)
  • Il ritornello sarà formattato in grassetto
  • Le sezioni speciali come intro, outro, bridge saranno evidenziate in corsivo

Note aggiuntive

  • Lo script chiederà anche un titolo, un link YouTube (opzionale) e i numeri di riferimento nei libretti
  • Il link YouTube verrà automaticamente convertito in un embed
  • Il nome del file HTML verrà generato automaticamente dal titolo
